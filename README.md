# PHP Web Programming Practice

This repository contains exercises and projects completed during the PHP Web Programming course.

## Contents
- In-class PHP exercises
- Practice assignments
- Form handling with GET and POST
- A final course project

## Final Project

The final project is a web application for **online football field booking and team matching**.

The system allows users to:
- Register and log in
- View available football fields
- Book football fields online
- Find teammates or join football teams

This project was developed using PHP and MySQL to practice server-side web development and database integration.
